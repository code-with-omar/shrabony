Text Formatting Tags

1. Bold Text - <strong>

Example: <strong>This text is bolded.</strong>

2. Italic Text - <em>

Example: <em>This text is italicized.</em>

3. Underlined Text - <u>

Example: <u>This text is underlined.</u>

This text is underlined.

4. Highlighted Text - <mark>

Example: <mark>This text is highlighted.</mark>

This text is highlighted.

5. Smaller Text - <small>

Example: <small>This text appears smaller.</small>

This text appears smaller.
